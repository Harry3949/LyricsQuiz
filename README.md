# 歌詞クイズ（Lyrics Quiz）
好きなアーティストの歌詞を当てる、JavaScript製のクイズアプリケーションです。
## 🚀 プロジェクト概要
このアプリは、主要なアーティストの楽曲から歌詞の一部を引用し、その曲名を当てるクイズゲームです。
JavaScriptによる動的なUI操作と、JSONデータを用いたコンテンツ管理の学習を目的として制作しました。
## ✨ 主な機能
- **アーティスト選択**: 複数の人気アーティストからクイズを選択可能。
- **難易度設定**: かんたん・ふつう・むずかしいの3段階（制限時間の変動）。
- **スコアシステム**: 正解数に加え、解答スピードに応じたタイムボーナスを算出。
- **背景の動的変更**: 選択したアーティストに合わせてUIのテーマカラーや背景が変化。
- **楽曲リスト表示**: クイズ終了後、出題された曲を一覧表示し、公式動画（YouTube等）へアクセス可能。
  
## 🛠 使用技術
- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **Data**: JSON (歌詞・リンク情報の管理)
## 💡 こだわったポイント
- **没入感のあるデザイン**: アーティストごとに背景画像を切り替え、ファンが楽しめる雰囲気を演出しました。
- **データ設計**: `lyrics.json` で問題を一括管理しており、新しいアーティストや楽曲の追加が容易な設計にしています。
- **ユーザー体験（UX）**: ローディングバーによる制限時間の可視化や、解答後のフィードバックアニメーションを実装しました。
