# 歌詞クイズ アプリケーション 起動手順書

このドキュメントでは、「歌詞クイズ」アプリケーションをローカル環境で起動し、使用するための手順を説明します。

---

## 1. 概要
このアプリケーションは、好きなアーティストの歌詞の一部から曲名を当てるクイズゲームです。ブラウザ上で動作します。

## 2. 必要な環境
- **Webブラウザ**: Google Chrome, Microsoft Edge, Firefox などの最新版。
- **ローカルサーバー環境**: このアプリは `lyrics.json` ファイルを読み込む（fetch）ため、直接 `index.html` ファイルをブラウザで開くと、セキュリティ制限（CORSエラー）により動作しません。以下のいずれかの方法で起動してください。

## 3. 起動方法

### 方法A: VS Code の「Live Server」拡張機能を使用する場合（推奨）
1. Visual Studio Code で本プロジェクトのフォルダ（`在永陽斗`）を開きます。
2. `index.html` を開きます。
3. 画面右下の「**Go Live**」をクリック、または `index.html` を右クリックして「**Open with Live Server**」を選択します。
4. 自動的にブラウザが立ち上がり、アプリが起動します。

### 方法B: npx (Node.js) を使用する場合
1. ターミナルまたはコマンドプロンプトを開き、プロジェクトのディレクトリに移動します。
2. 以下のコマンドを実行します：
   ```bash
   npx serve .
   ```
3. 表示されたURL（例: `http://localhost:3000`）をブラウザで開きます。

## 4. アプリケーションの操作方法
1. **アーティスト選択**: 画面に表示されているアーティストのボタンをクリックします。
2. **難易度選択**: 「かんたん」「ふつう」「むずかしい」から選んでクリックします。
3. **クイズ回答**: 表示された歌詞を見て、正しい曲名を選択肢（3択）から制限時間内にクリックします。
4. **結果確認**: 10問終了後、スコアが表示されます。「出題された曲を見る」ボタンから正解の曲を確認（リンクがあれば試聴可能）できます。

## 5. 注意事項・トラブルシューティング

### 画面が「JSONの読み込みに失敗しました」と表示される / 動作しない
- **原因**: 起動方法が間違っています。ファイルを直接（`file:///...`）開かず、必ず上記の**ローカルサーバー**経由で開いてください。
- **解決策**: 手順3に従って起動してください。

### 音声（BGM）が流れない
- **現状**: `index.html` 内で `bgm/floweranddancer.mp3` を指定していますが、プロジェクトディレクトリ内に `bgm` フォルダが存在しない場合があります。
- **解決策**: 必要に応じて `bgm` フォルダを作成し、該当のMP3ファイルを配置してください。

### 画像が表示されない
- **確認事項**: `image` フォルダ内に以下のファイルが含まれているか確認してください。
  - `リイシューねこちゃん.jpg`
  - `Mrs.GREEEN APPLE.jpg`
  - `嵐.jpg`
  - `まふまふ.jpg`
  - `deco27.jpg`
  - `Kanaria.jpg`
